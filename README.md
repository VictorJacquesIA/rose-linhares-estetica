# Rose Linhares Estética

Landing page · São José, SC

## Comandos

```bash
npm install
npm run dev      → http://localhost:5173
npm run build    → /dist
npm run preview  → preview do build
```

## Deploy

Importar repositório no vercel.com → Deploy automático.

## Substituições pendentes

- /assets/images/hero.webp → foto real da Rose
- /assets/images/avatar-*.webp → fotos reais das clientes
- tracking.js → adicionar IDs reais de GA4 e Meta Pixel
- vercel.json → domínio final quando disponível
